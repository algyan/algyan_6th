## <a name="1-0">1. 部品のはんだ付け</a>

部品のはんだ付けは、高さが低く、小さな部品から、また同じ種類、型式のものをまとめて行っていくと間違いが少なくできます。  
目安として、パーツリストの各部品欄左下の番号順(Set Aは、①～⑬、Set Bは①～㉔)に部品をはんだ付けしていく形になります。  
また、今回の基板は金メッキ基板のため、チップ部品のはんだ付けには必ずランド(※)に予備はんだが必要になります。  

※)ランドとは、部品の取り付け及び接続に用いる導体パターンのことです。  
　表面実装用の場合は「パッド」と呼ぶこともあります。  

### <a name="1-1">1-1. Set A(お手軽)</a>
#### <a name="1-1-1">1-1-1. フルカラーRGB LED(LED4)</a>
フルカラーRGB LED(LED4)には、方向があります。  
下の写真のようにカソード側にマークがついています。  
![LED4](img/LED4.png)

フルカラーRGB LED(LED4)のはんだ付けにおいては、6つのランド(※)の内、1つのランドに予備はんだを盛ります。  
予備はんだを盛るランドは、コーナーやその近くでGNDのようなベタパターンと繋がっていない方に行います。  
(ベタパターンと繋がっているランドには、はんだごての熱が伝わりにくく、はんだがのりにくいので避けます。)  
![LED4a](img/LED4a.png)

カソードがLED4の「4」の文字の方に来るようにはんだ付けします。  
最初は、予備はんだを付けたランドにLEDをピンセットで摘まんで、位置を合わせて予備はんだをこてで溶かして取り付けます。  
![LED4b](img/LED4b.png)

次に対角のランドのはんだ付けをします。  
はんだ付けのコツは、まずこてで基板のランドを1、2秒ぐらい温めます。
その後、はんだを流してつけます。  
![LED4c](img/LED4c.png)

最後に残りのランドのはんだ付けをします。
![LED4d](img/LED4d.png)

#### <a name="1-1-7">1-1-7. Wi-Fiモジュール ESP32DevkitC-32E用ソケット(U5)</a>
ESP32-DevkitC-32Eは、直接はんだ付けせずに、シングルピンソケット(低メス)を使って取り付け、取り外しができるようにします。  
シングルピンソケット(低メス)は20ピンなので、**1ピンを折って切り取り**それぞれ19ピンにしておきます。

以下の写真は、シングルピンソケット(低メス)のはんだ付けが終わった状態です。
![U5](img/U5.png)

少し堅いのですが、頑張って嵌めてしまいましょう。
![U5a](img/U5a.png)

2ピンのはんだ付けが終わったところで、基板からソケットが浮いていないか、曲がっていないかを確認します。

![U5b](img/U5b.png)

はんだ付けでピン間をショートさせてしまった場合は、はんだ吸い取り線などで余分なはんだを取り除いてフラックスを塗り、ピン間をこてで切るようにするとショートを解消できたりします。

#### <a name="1-2-3a">1-2-3a. チップLED(LED1、LED2、LED3)</a>
チップLED(LED1、LED2、LED3)は、フルカラーRGB LED(LED4)と **排他(※)** で実装します。  
※)チップLED(LED1、LED2、LED3)とフルカラーRGB LED(LED4)を両方実装すると、**電気的な特性から正常に光りません**。このため、片方のみを実装します。  

またチップLED(LED1、LED2、LED3)には、方向があります。  
下の写真のようにカソード側にマークがついています。  
![LED1](img/LED1.png)

チップLED(LED1、LED2、LED3)のはんだ付けにおいては、片側のランドに予備はんだを盛ります。  
予備はんだを盛るランドは、GNDのようなベタパターンと繋がっていない方に行います。  
(ベタパターンと繋がっているランドには、はんだごての熱が伝わりにくく、はんだがのりにくいので避けます。)  
カソードが基板の外側に来る(写真の)ようにはんだ付けします。  
最初は、予備はんだを付けたランドにLEDをピンセットで摘まんで、位置を合わせて予備はんだをこてで溶かして取り付けます。  
![LED1-3a](img/LED1-3a.png)

#### <a name="1-2-3b">1-2-3b. フルカラーRGB LED(LED4)</a>
フルカラーRGB LED(LED4)にも、方向があります。  
下の写真のようにカソード側にマークがついています。  
![LED4](img/LED4.png)

フルカラーRGB LED(LED4)のはんだ付けにおいては、6つのランド(※)の内、1つのランドに予備はんだを盛ります。  
予備はんだを盛るランドは、コーナーやその近くでGNDのようなベタパターンと繋がっていない方に行います。  
(ベタパターンと繋がっているランドには、はんだごての熱が伝わりにくく、はんだがのりにくいので避けます。)  
![LED4a](img/LED4a.png)

カソードがLED4の「4」の文字の方に来るようにはんだ付けします。  
最初は、予備はんだを付けたランドにLEDをピンセットで摘まんで、位置を合わせて予備はんだをこてで溶かして取り付けます。  
![LED4b](img/LED4b.png)

次に対角のランドのはんだ付けをします。  
はんだ付けのコツは、まずこてで基板のランドを1、2秒ぐらい温めます。
その後、はんだを流してつけます。  
![LED4c](img/LED4c.png)

最後に残りのランドのはんだ付けをします。
![LED4d](img/LED4d.png)

#### <a name="1-2-18">1-2-18. ピンヘッダ(CN1、CN9、CN3、CN4)</a>
各コネクタ用のピンヘッダを同様にはんだ付けします。  
コネクタCN2、CN9は、モータドライバDRV8835のモータ駆動用コネクタなので、モータ駆動をしない場合は取り付け不要です。  
コネクタCN3はサーボモータ駆動用コネクタなので、サーボモータを駆動しない場合は取り付け不要です。  

各コネクタのはんだ付けのときも各1本のピンだけはんだ付けが終わったところで、基板からピンヘッダが浮いていないか、曲がっていないかを確認します。  
浮いていたり、曲がっていたらこのときに修正します。  
1本のピンのはんだ付けに問題なければ、残りのピンのはんだ付けをします。  
以下の写真はSet A(お手軽)でのものですが、同じように実装します。
![CN1-9](img/CN1-9.png)

コネクタCN2、CN9をはんだ付けして、モータドライバDRV8835を取り付け、モータ駆動をする場合は、さらに基板裏側のSJ3の片側をはんだ付けしてショートさせる必要があります。  
以下の写真の場合は、CN9からモータ駆動用外部電源を供給する場合のパターンショートの例です。
